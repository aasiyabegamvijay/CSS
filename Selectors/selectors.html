Selectors

Types of Selectors

  1. Class
  2. Id
  3. Universal
  4. Group
  5. Element
  6. Descendent

1. Element Selector

An element tag in HTML defines the structure and content of a webpage. It usually consists of an opening tag, content, and a closing tag (e.g. Content). Each element serves a specific purpose, such as headings, paragraphs, images, or links.

2. Id Selector

An ID selector in HTML targets a single unique element using its ID attribute to apply specific CSS styles or behavior.

3. Class Selector

A class selector in HTML targets elements with a specific class name, allowing shared styling across multiple elements using CSS.

4. Universal Selector

A universal selector in HTML, represented by an asterisk (*), is used in CSS to apply styles to all elements on a webpage. It helps maintain consistency by setting common properties like margin, padding, or font across the entire document, ensuring a uniform appearance and simplifying global styling.

5. Group Selector

A group selector in HTML allows applying the same CSS styles to multiple elements by separating their selectors with commas.

6. Descendent Selector

A descendant selector in HTML is used to apply CSS styles to elements that are nested within a specific parent element. It helps in targeting only the child or inner elements without affecting others. The selector is written by separating parent and child selectors with a space (e.g., `div p {}` ).
